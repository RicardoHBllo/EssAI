1.- ¿Como logramos esto?

Teniendo a disposición equipos de computo del equipo, conceptos de programación web, también tenemos a nuestro alcance apis de terceros para lograr que funciones nuestro software al igual que la dedicación de nuestro equipo para que esto salga a flote

2.-Propósito

El propósito es generar un ensayo científico (con entrada en ingles) dado por un formulario al usuario sobre que tema quiere tratar, esto interpretado y generado por IA, dándole al usuario la libertad de cuanto quiere profundizar sobre el tema, para finalizar con un archivo descargable en formato pdf al usuario. Cabe mencionar que también cuenta con un sistema anti plagio para no generar coincidencias con otros escritos ya hechos.

3.-Lo que usamos (materiales o disposición como api’s)

Lo utilizado fue: Api de ChatGpt (openAI) smoothie (para evitar plagios) (Generar/Insertar imagenes en el escrito) Lenguajes de programación: JavaScript HTML5 CSS Php

Desarrollo de funcionalidad

Del 24 de Octubre al 8 de Noviembre de 2023

El desarrollo fue lo que mas represento reto del proyecto, debido a que también necesitábamos invertir en una api de una inteligencia artificial para generar los ensayos y también en pensar como implementarla, pero al mismo tiempo en como seria la interacción usuario a software, por lo que optamos en la creación de un formulario para procesar los datos ingresados por el usuario y genere un archivo pdf que se descargue, con todo lo anterior mencionado, fue desarrollado con javascript, lo que facilito la integración de la api de ChatGPT con el software EssAIs y funcione en conjunto para brindar una experiencia eficaz. Pero ¿como funciona?, fácil, utiliza la función fetch para realizar una solicitud GET a la API de JSONPlaceholder y obtener datos de usuarios. Luego, convierte la respuesta a formato JSON y crea dinámica mente una lista (ul) en el DOM que muestra el nombre y el correo electrónico de cada usuario. Después utiliza la función getCompletion para realizar una solicitud POST a la API de OpenAI. Luego, configura un evento de clic en el botón con el ID "generarPDF" que recoge datos del formulario, crea un prompt y utiliza la función getCompletion para obtener una respuesta de la API de OpenAI. El texto generado se muestra en el elemento con el ID "app".
